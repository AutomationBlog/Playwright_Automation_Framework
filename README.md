# Playwright + Cucumber TypeScript E2E Framework

This is a sample Playwright + Cucumber (Gherkin) automation framework using TypeScript.

Quick start

1. Install dependencies:

```bash
npm install
npm run install-playwright
```

2. Run the sample Amazon add-to-cart scenario:

```bash
npm test
```

Reports are generated at `reports/` (JSON + HTML). Screenshots are saved to `reports/screenshots`.
